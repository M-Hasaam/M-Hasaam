# Muhammad Hasaam | Software Engineer Portfolio

A professional, high-performance portfolio website built with clean architecture and modern web standards. This site showcases my work in Software Engineering, Flutter development, and Competitive Programming.

## 🚀 Overview

This portfolio is designed to present technical expertise and project achievements in a minimalist, accessible format. It serves as a central hub for my professional profile, including project repositories, certifications,  and contact information.

## ✨ Key Features

- **Premium Branding**: Custom Golden 'T' icon and full name brand title for a prestigious look.
- **Custom Favicon**: Bespoke 'TS' monogram SVG favicon for professional browser tab presence.
- **Elegant Light Theme**: Default-to-light mode for a clean, accessible first impression.
- **Dynamic Projects**: Integration with project data for seamless updates.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive UI**: Subtle animations and transitions to enhance user experience without distraction.
- **Theme Support**: Consistent user preferences with both Light and Dark modes.

## 🛠️ Technology Stack

- **Core**: HTML5, CSS3 (Modern Flex/Grid), Vanilla JavaScript
- **Styling**: Modular CSS architecture for maintainability
- **Icons**: Font Awesome 6.5.1
- **Typography**: Google Fonts (Montserrat & Cormorant Garamond)

## 📁 Repository Structure

```text
.
├── assets/             # Images, icons, and document assets (Resume, CV)
├── css/                # Modular CSS files grouped by function
├── js/                 # Core logic and interactive components
├── json/               # Data files for projects and achievements
└── index.html          # Main entry point
```

## 🛠️ Getting Started

### Prerequisites

No build tools are required. The project runs directly in any modern web browser.

### Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/M-Hasaam/Portfolio.git
   ```
2. **Configure Assets**:
   Place your professional documents in the `assets/` directory:
   - `assets/resume.pdf`
   - `assets/cv.pdf`
3. **Launch**:
   Open `index.html` in your browser or serve via a local development server for the best experience.
